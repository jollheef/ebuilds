# How to use

    $ sudo eselect repository enable jollheef-overlay

or

    $ sudo layman -a jollheef-overlay

or

    $ sudo layman -o https://github.com/jollheef/jollheef-overlay/raw/master/overlay.xml -f -a jollheef-overlay
